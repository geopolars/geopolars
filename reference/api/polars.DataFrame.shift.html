polars.DataFrame.shift#

DataFrame.shift(periods: int) → DF#

Shift values by the given period.

Parameters:
periods

Number of places to shift (may be negative).

See also

shift_and_fill

Examples

>>> df = pl.DataFrame(
...     {
...         "foo": [1, 2, 3],
...         "bar": [6, 7, 8],
...         "ham": ["a", "b", "c"],
...     }
... )
>>> df.shift(periods=1)
