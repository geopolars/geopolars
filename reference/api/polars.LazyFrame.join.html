polars.LazyFrame.join#

LazyFrame.join(other: LazyFrame, left_on: str | pli.Expr | Sequence[str | pli.Expr] | None = None, right_on: str | pli.Expr | Sequence[str | pli.Expr] | None = None, on: str | pli.Expr | Sequence[str | pli.Expr] | None = None, how: JoinStrategy = 'inner', suffix: str = '_right', allow_parallel: bool = True, force_parallel: bool = False) → LDF#

Add a join operation to the Logical Plan.

Parameters:
other

Lazy DataFrame to join with.

left_on

Join column of the left DataFrame.

right_on

Join column of the right DataFrame.

on

Join column of both DataFrames. If set, left_on and right_on should be None.

how{‘inner’, ‘left’, ‘outer’, ‘semi’, ‘anti’, ‘cross’}

Join strategy.

suffix

Suffix to append to columns with a duplicate name.

allow_parallel

Allow the physical plan to optionally evaluate the computation of both DataFrames up to the join in parallel.

force_parallel

Force the physical plan to evaluate the computation of both DataFrames up to the join in parallel.

See also

join_asof

Examples

>>> df = pl.DataFrame(
...     {
...         "foo": [1, 2, 3],
...         "bar": [6.0, 7.0, 8.0],
...         "ham": ["a", "b", "c"],
...     }
... ).lazy()
>>> other_df = pl.DataFrame(
...     {
...         "apple": ["x", "y", "z"],
...         "ham": ["a", "b", "d"],
...     }
... ).lazy()
